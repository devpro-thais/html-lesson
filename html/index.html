E este é um parágrafo , um dos elementos mais comuns em documentos HTML.

Existem diferentes níveis de título, do 1 ao 6:

Titulo nível 2

Titulo nível 3

Titulo nível 4

Titulo nível 5
Titulo nível 6

É possível enfatizar em uma palavra com o elemento em, dar ainda mais ênfase com o elemento strong, ou marcar uma palavra com o elemento mark. Embora existam os elementos i (itálico) e b (bold), eles não são mais recomendados, pois o HTML deve focar no conteúdo e semântica, deixando a estilização para o CSS.

Também podemos demarcar o código fonte com o elemento code, ou demarcar entrada de teclado com o elemento kbd.

Podemos forçar uma quebra de linha aqui
com o elem br.

Podemos usar sub para subscrito, como em H2O, ou sup para sobrescrito, como em x2

Imagens são inseridas com o elemento img.

A imagem pode ser hospedada em outro site.

O elemento a corresponde a um link para outro documento.

Links podem ser relativos ou absolutos, dependendo do formato do atributo href. Este é um link absoluto.
